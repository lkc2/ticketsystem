# 客服回報管理系統 (Customer Ticket System)

## 功能特色
- [x] 用戶 CRUD（註冊/查詢/更新/刪除）
- [x] 客服單 CRUD
- [x] 回覆 CRUD
- [x] RESTful API 設計
- [x] Swagger 文件/測試
- [x] 支援 Postman 測試

## 技術棧
- Java 21
- Spring Boot 3.5.3
- MyBatis-Plus 3.5.12
- MySQL 8.x
- Swagger
- Postman
- 前端：Vue3

## 主要 API 路徑：
- /api/users 用戶管理
- /api/tickets 客服單管理
- /api/replies 回覆管理
