# 客服回報管理系統 (Customer Ticket System)

## 專案簡介
- 本專案為客服回報管理系統的前端，支援用戶註冊、客服單管理、回覆紀錄管理等功能，並串接後端 RESTful API。

## 技術棧
- Vue 3
- Vite
- Element Plus
- Pinia
- Axios

## 功能列表：
- 登入
- 用戶管理（新增、刪除、查詢）
- 客服單管理（新增、查詢、搜尋、刪除）
- 客服回覆（新增、查詢、刪除）
- 美觀 UI/可響應設計
- 介面友善互動
